**launch**
- 여러 ROS2 노드, 파라미터, 구성 파일들을 하나의 명령어로 동시에 실행하고 관리하는 스크립트 파일
- .py, .xml, .yaml 형식을 사용
- 반복문, 조건문 등 로직 사용이 가능한 .py를 가장 많이 사용
- .xml은 구조가 직관적이어서 복잡한 로직이 필요 없다면 좋음


**Substitutions**
- launch 파일이 실행될 때 값이 동적으로 계산되어 치환되는 표현식
- launch를 실행할 때 불필요하게 고정값을 사용하지 않고 필요한 값을 동적으로 계산, 적용 가능
- 복잡한 환경의 로봇 시스템 구축에 유리


**event handler**
- 런치 시스템 내에서 특정 이벤트가 발생했을 때, 미리 정의된 동작(Action)을 수행하게 하는 기능
